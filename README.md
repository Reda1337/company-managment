# Company Management System

Spring Boot application for managing company operations including employees, departments, and projects.

## 🚀 Features

- **User Management**: JWT authentication with role-based access control
- **Employee Directory**: Complete CRUD operations with pagination and filtering
- **Department Management**: Organizational structure with employee assignments
- **Project Tracking**: Project and task management with status tracking
- **Activity Logs**: Comprehensive audit trail for all operations

## 🛠️ Tech Stack (100% Free & Open Source)

- **Backend**: Spring Boot 3.2, Java 17
- **Database**: PostgreSQL 15
- **Security**: Spring Security with JWT
- **Documentation**: OpenAPI 3.0 (Swagger)
- **Containerization**: Docker & Docker Compose
- **Build Tool**: Gradle
- **Database Tools**: Adminer, pgAdmin

## 📋 Prerequisites

All tools are free:
- Java 17 or higher (OpenJDK recommended)
- Docker & Docker Compose
- Git
- IDE: IntelliJ IDEA Community Edition
